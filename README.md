**Diploma year two. Full stack by trade, pixel perfectionist by nature.**

I build complete products — schema design, REST APIs, and the interface people actually touch. My stack leans MERN but I reach for C++ when I need raw control, Kotlin when I'm shipping Android, and WebGL when I want to make something no one's seen before.

Most UIs settle too early. I don't. Every project gets pushed until the spacing is right, the animations feel earned, and nothing looks like it came from a template. I care about how things work just as much as how they look — clean code on the backend, obsessive detail on the front.

Currently at **Netaji Subhas University** (Diploma, 2024–2027). Open to internships, freelance, or anything worth building.
